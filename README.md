# Kang

Kang is a lightweight VM substrate for running Kubernetes clusters on bare metal.

It provides a simple infrastructure layer for provisioning and managing VM-backed Kubernetes nodes without the operational complexity of traditional IaaS platforms.

## Goals

- Lightweight VM provisioning on bare metal
- Kubernetes-focused infrastructure
- Cluster API compatibility
- Simple host and network management
- Production-grade operational model
